SYNTAX SURVIVOR

What's more fun than playing computer games? Games that test your knowledge of programming syntax of course!

How to play

  1. Use the up and down arrow keys to move the player
  2. Collect syntax tokens with the correct syntax to gain points
  3. If you collect a token with the incorrect syntax, you lose a life
  4. If you miss token with the correct syntax, you lose a life
  5. The game is over when you lose all lives
  6. It's that simple! Click the button below to start the game
